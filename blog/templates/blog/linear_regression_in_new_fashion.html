Basic difference between modern technology paradigms

In this blog post, I will demonstrate what exactly is data science, skill set required to be a data scientist and the brief comparsion with related fields like Machine Learning, Artifical Intelligence, Deep Learning...

Data Science

“One of the greatest challenges for 21st-century science is how we respond to this new era of data-intensive science. This is recognized as a new paradigm beyond experimental and theoretical research and computer simulations of natural phenomena—one that requires new tools, techniques, and ways of working.”
— Douglas Kell, University of Manchester
Data Science involves collection, prepration, analysis, visualization, management, and preservation of large collection of information. The goal is to extract insights from data, predict different outcomes and sometimes automate the processess like data collection, cleaning and archieving.

Skills Needed To Become a Data Scientist

Following are some important attributs and skills .

  • Programming Languages
  • Mathematics
  • Domain Knowledge

Programming Languages

A data scientist should know atleast any one the essential languages Python or R, As the language of choice and rate of adaptation is moving towards Python. I will be mostly focussing on Python in this blog post, there are also some other languages in use like Scala, Clojure, Java, Octave and Julia.

To be really successfull data scientist, the programming skills needed are:

  • Dealing with large volumes of data, working with real time data, cloud computing, big data technologies, unstructured data, databases.
  • Working knowledge of various scientific numerical libraries like Numpy, SciKit learn, TensorFlow, PyTorch, Keras.
  • Knowledge of statistical models like regression, optimization, clustering, decision trees, random forests etc.
