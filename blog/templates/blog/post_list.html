Deploying Django App with Apache and Wsgi

Deploying django with apache and mod_wsgi is a simple way to get django project into production. The mod_wsgi is an Apache HTTP Server module that provides WSGI compliant interface for hosting Python based web applications under Apache. This blog is a guide to deploy django app on Amazon ec2 instance using git. It also explains about mapping ec2 elastic IP with godaddy DNS.
